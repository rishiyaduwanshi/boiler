Update Boiler to latest version

### Synopsis

Update Boiler CLI to the latest version.

Downloads and installs the latest release from GitHub.

```
bl self update [flags]
```

### Examples

```
  # Update to latest version
  bl self update
```

### Options

```
  -h, --help   help for update
```
